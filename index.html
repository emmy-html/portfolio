The LOLCAT Clock Project gave me a variety of tasks to translate into JavaScript. I had to create a clock that displayed the time, while also displaying a different message and image depending on the time of day. By breaking these needs down into pseudocode, I was able to write the appropriate JavaScript functions to bring it all together; then finally connect it to the HTML and CSS that was provided to me.
